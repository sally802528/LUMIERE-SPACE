線上圖庫連接教學

您可以透過網址參數自動載入 GitHub 上的圖庫。格式如下：

/?load=使用者名稱/儲存庫名稱/

範例:/?load=sally802528/TEST_online_LS/

請確保儲存庫根目錄包含一個 manifest.json 檔案，其中定義了所有媒體資源的連結與類型。
